# 📊 Sales & Operations Dashboard

This project showcases an **interactive Power BI dashboard** designed to analyze
sales and operations in an agile and effective way.

## 🔍 Key features

- Segmentation by country and year.
- KPIs: **total sales, %MoM and %YoY** variance.
- Line chart for monthly sales trends.
- Top 10 best-selling products.
- Comparative sales chart by country.

## 🛠️ Tools used

- Power BI
- Excel (original data sources)
